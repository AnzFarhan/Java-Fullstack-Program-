## ▶️ How to Run
1. Clone this repo:
    - git clone https://github.com/<your-username>/<your-repo-name>.git
    - cd <your-repo-name>

2. Open backend projects in IntelliJ IDEA.
    - Build & run using Maven/Gradle.

3. Open frontend projects in VS Code or your preferred editor.
- Start with:
    - npm install
    - npm start
